# おこめ診断

食感・甘み・食べ方の好みから、おすすめのおこめの銘柄を診断する小さなアプリです。

## 画面の開き方

VSCodeでこのディレクトリを開くことは必須ではありません。

`localhost:3001` で確認したい場合は、ターミナルでこのディレクトリに移動してからサーバーを起動します。

```bash
cd /Users/mai/Projects/okome_shindan
python3 -m http.server 3001
```

そのあと、ブラウザで次のURLを開きます。

```text
http://localhost:3001
```

## 使い方

1. 食感・甘み・食べ方の好みを選ぶ
2. 「診断する」を押す
3. おすすめのお米の銘柄と理由を確認する
